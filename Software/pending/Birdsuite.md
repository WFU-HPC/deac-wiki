## Installation

### Affymetrix Power Tools

  - Already installed cluster wide\[2\] -- use appropriate module:

`    [username@headnode]$ module load apt/1.10.2`

### mclust R module

  - Install into own directory
  - Download tarball
  - At commandline:

`    [username@headnode]$ R CMD INSTALL mclust-x.yy.z.tar.gz`
