Portland Group, Inc. (PGI) Cluster Development Kit 12.1[1] is available on the cluster.

To use the PGI compiler suite, load one of these modules depending on your MPI requirements:

  $ module loadmodulename   wheremodulename is one of: compilers/pgi-12.1, compilers/pgi-12.1-mpi, compilers/pgi-12.1-mpi2, compilers/pgi-12.1-mvapich

Commands¶

These are some of the more commonly used commands in the PGI suite – most have a “-help” option that prints out usage information:

  • pgcc - C compiler

  • pgc++ - C++ compiler

  • pgfortran - Fortran compiler

  • pgdbg - debugger (will also debug MPI programs)
