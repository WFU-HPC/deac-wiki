Specifics¶

Use module:

    $ module load cuda/9

  • CUDA Installation Directory : /usr/local/cuda (for latest) CUDA Driver Version / Runtime Version : 9.2 / 9.1 CUDA Capability Major/Minor version number: 6.0

Job Submission¶

Support for GPU job submission is enabled the GRES (generic resource) configuration within SLURM.[4]

See SLURM:Job_Script_Templates#Job_Script_for_GPU for how to request GPU resources.
